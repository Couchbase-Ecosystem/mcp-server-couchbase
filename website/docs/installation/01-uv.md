# Install from PyPI

The pre-built [PyPI package](https://pypi.org/project/couchbase-mcp-server/) is the easiest way to get started. No manual installation is needed — `uvx` runs the package directly.

## Prerequisites

- **Python 3.10+** installed
- **[uv](https://docs.astral.sh/uv/)** installed

## Verify Installation

```bash
uvx couchbase-mcp-server --version
```

## MCP Client Configuration

When configuring an MCP client, use this command format:

```json
{
  "mcpServers": {
    "couchbase": {
      "command": "uvx",
      "args": ["couchbase-mcp-server"],
      "env": {
        "CB_CONNECTION_STRING": "couchbases://your-connection-string",
        "CB_USERNAME": "username",
        "CB_PASSWORD": "password"
      }
    }
  }
}
```

:::tip
If you have other MCP servers configured, add the `couchbase` entry to the existing `mcpServers` object.
:::
