# Windsurf Editor

Follow these steps to use the Couchbase MCP Server with [Windsurf Editor](https://windsurf.com/).

## Setup

1. Install [Windsurf Editor](https://windsurf.com/download) on your machine.

2. Navigate to **Command Palette > Windsurf MCP Configuration Panel** or **Windsurf - Settings > Advanced > Cascade > Model Context Protocol (MCP) Servers**. See the [official documentation](https://docs.windsurf.com/windsurf/cascade/mcp#adding-a-new-mcp-plugin) for details.

3. Click on **Add Server** and then **Add custom server**. Add the Couchbase MCP Server [configuration](/docs/getting-started/installation).

4. Save the configuration.

5. You will see **couchbase** listed in MCP Servers under Advanced Settings. Refresh to confirm the server is enabled.

6. You can now use the Couchbase MCP Server in Windsurf Editor to query your Couchbase cluster using natural language and perform CRUD operations on documents.

For more details, refer to the [Windsurf MCP documentation](https://docs.windsurf.com/windsurf/cascade/mcp).
